# Demo

* demo is in Ireland region (eu-west-1)
* users should be added to iam identity center
* go to QBusiness
  * click on application
  * Manage User access
  * update subscription of sandbox.admin to Q Business Lite
* For the demo, you can show the different sources that can be configured
* You can show the S3 Bucket where the data resides
* log again on SSO Portal
  * go on Application Tab
  * click on QBusiness

* Type
```
What is anycompany?
```

* a response should appear
* in tab Sources, you should see the file that was in the s3 bucket in reference
