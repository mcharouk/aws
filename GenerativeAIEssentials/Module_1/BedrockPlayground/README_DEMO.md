# Single Text Prompt

* Go to TextPlayground
* Select simple Prompt mode
* Select Nova Micro

```
Write a 150-word product description for a fictional jazz album titled "Midnight Serenade" by the artist "Luna Jazz Quartet". Highlight the overall mood, standout tracks, and instrumental performances
```

# Image creation

* Clear the prompt
* Ask genAI to create a prompt to create an image

```
Create a prompt for an AI image generator to design an album cover for "Midnight Serenade" by Luna Jazz Quartet. The album cover should capture the essence of late-night jazz and the album's mood. Include specific details about colors, imagery, and composition. 

The prompt should have the following elements: 

1. Introductory directive. For example: "design an album cover for "Midnight Serenade" by Luna Jazz Quartet with the following elements:" 
2. bullet list of short image directives (each bullet upto 5 words) to capture the style, mood, color, imagery, etc. 

The prompt is less than 200 words.
```

* Take the output of this prompt and go to image playground
* select Nova Canvas
* copy prompt and generate the images

# Chat Prompt

* Select Chat mode
* Select Nova Lite
   
```
You are an AI music recommendation system. I will provide my music preferences, and you will suggest three albums I might enjoy, explaining why for each. Respond in a conversational tone. My input: "I enjoy melodic indie rock with female vocalists and electronic elements."
```

* After response , copy following

```
Thanks for the suggestions! I particularly like the sound of the second album. Can you recommend more albums similar to that one, but with a bit more of an upbeat tempo?
```
